Capability

This web application is capable of analysing Available datasets on COVID 19 and cleans the data, transforms and also applies a slicing algorithm for dividing the data set into a training and testing sets. These sets are then fed into a machine learning algorithm to magnify trends in the virus trends.

The Program is based on a deep learning neural network which is able to learn trends in the dataset and lay down the foundations for prediction and classification analysis which is showcased through this program.

Diagnosis spectrum of COVID-19

This program will allow medical specialists to detect whether a person has coronavirus using classification analysis using DNN. The application incorporates another model for modelling the number of deaths which are predicted based on current death rate of another country. The Map functionality shows the spread of the virus across various countries and shows what genome length and which strain is most prevalent across the countries. The minimum and maximum age of infection is also found using this algorithm. The diabetes classification DNN is capable of detection of diabetes based on various core morbidity factors.
